# WTFChat
just chat in every mode :v

command
- !chat [text]
- !chatset [mode]
<br>*default mode always set to Say

| Mode |
| ------------------------ |
| s = Say |
| p = Party |
| g = Guild |
| a = Area |
| t = Trade |
| greet = Greet |
| pn = Party-notice |
| rn = Raid-notice|
| e = Emote |
| c = Global |
| r = Raid |

*not support Private ,Megaphone, Guild Advisor
